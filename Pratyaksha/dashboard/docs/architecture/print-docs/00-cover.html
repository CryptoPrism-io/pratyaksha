Technical Architecture Documentation

Version 1.0 • February 2026

Product Overview

AI-powered journaling platform that analyzes emotional patterns, provides personalized insights, and helps users understand their cognitive and emotional landscape through data-driven reflection.

Component Description
4-Agent AI Pipeline Intent → Emotion → Theme → Insight sequential processing
RAG Chat Semantic search with pgvector, two-pass generation
Personalization Soul Mapping (5 topics), Life Blueprint (vision/goals/levers)
Database PostgreSQL 16 + pgvector, 13 tables, Drizzle ORM

Tech Stack

Frontend
  • Framework: React 18 + Vite (Port 5173)
  • State Management: TanStack Query (5min stale, 30s polling)
  • Styling: Tailwind CSS + shadcn/ui components
  • Charts: Recharts for data visualization
  • Authentication: Firebase Auth with JWT tokens
Backend
  • Framework: Express + TypeScript (Port 3001)
  • Database: PostgreSQL 16 on Cloud SQL (34.55.195.199)
  • ORM: Drizzle ORM with postgres driver
  • AI: LangChain + OpenRouter (gpt-4o, gpt-4o-mini)
  • Vectors: pgvector v0.8.1 (1536d embeddings)
  • Speech: Groq SDK (Whisper + llama-3.3-70b)
AI Models
Model Purpose
gpt-4o Intent Agent, Insight Agent (quality)
gpt-4o-mini Emotion Agent, Theme Agent (fast & cheap)
text-embedding-3-small Vector embeddings (1536 dimensions)
whisper-large-v3 Speech-to-text (Groq)
